========================================
POS AppSheet - APK Build Tools
========================================

CARA PENGGUNAAN:

1. PASTIKAN SUDAH INSTALL:
   - Java JDK 17
   - Android SDK Command Line Tools
   - Atau Android Studio (untuk SDK)

2. SET ENVIRONMENT VARIABLES:
   - JAVA_HOME = C:\Program Files\Java\jdk-17
   - ANDROID_HOME = C:\Users\[user]\AppData\Local\Android\Sdk

3. JALANKAN:
   - Klik 2x build-apk.bat
   - Atau buka CMD di folder ini, ketik: build-apk.bat

4. APK AKAN ADA DI:
   android-project\app\build\outputs\apk\debug\app-debug.apk

5. INSTALL KE HP:
   - Hubungkan HP via USB
   - Aktifkan USB Debugging
   - Jalankan install-adb.bat
